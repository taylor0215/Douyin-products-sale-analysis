# Douyin Products Sale Analysis  
**電商平台品牌與商品類別營收分析專案**  
使用 **SQL + Python + Tableau** 進行模擬電商平台銷售資料分析，探索平台主要營收來源、品牌定位與商品類別結構。
## 📌 Project Overview 專案簡介
本專案以模擬電商平台銷售資料為基礎，透過資料清理、聚合分析與視覺化呈現，分析平台營收主要來自哪些品牌與商品類別，並進一步觀察不同品牌的營收模式與平台營收結構特徵。

專案目標包括：

- 找出平台的主要營收來源品牌
- 分析不同商品類別對整體 GMV 的貢獻
- 區分品牌的營運模式（高價值 / 流量 / 核心 / 小眾）
- 練習從資料查詢、整理、分析到視覺化呈現的完整流程

---

## Business Questions 分析問題

本專案主要回答以下問題：

1. 平台營收主要集中在哪些品牌？
2. 哪些商品類別對整體營收貢獻最大？
3. 不同品牌的營收模式有何差異？

## 📂 Dataset 資料說明

本專案使用模擬電商平台商品銷售資料，資料欄位包含：

- 商品名稱
- 品牌名稱
- 商品類別
- 銷量
- 營收（GMV）

> 本專案資料為模擬資料，僅作為資料分析與作品集展示用途。

Analysis Process 分析流程

### 1. 資料清理與前處理
- 檢查欄位格式與缺失值
- 整理商品、品牌與類別資訊
- 建立可分析之商品層級資料表

### 2. 建立核心指標
透過 SQL 與 Python 整理以下指標：

- GMV（Gross Merchandise Value）
- 銷量
- 平均單價
- 品牌 / 類別層級彙總結果

### 3. 商品與品牌分群觀察
依據 GMV 與銷量表現，將商品與品牌區分為不同營運類型，例如：

- 高 GMV、高銷量（核心品牌）
- 高 GMV、低銷量（高價值品牌）
- 低 GMV、高銷量（流量型品牌）
- 低 GMV、低銷量（小眾品牌）

### 4. 視覺化與商業解讀
使用 Tableau 製作互動式儀表板，從以下三個視角進行分析：

- 品牌營收排名
- 品牌定位分布
- 商品類別營收結構

## 📊 Key Findings 主要發現

### 1. 頭部品牌集中度高
平台營收明顯集中於少數頭部品牌，其中Apple為平台營收最高品牌，顯示平台營收結構具有集中化特徵。

### 2. 高客單價商品貢獻明顯
珠寶禮品類 為平台最大營收來源，反映高單價商品對整體營收具有顯著影響。

### 3. 品牌商業模式分層明顯
不同品牌在平台中的營收模式可區分為：

- 核心品牌：高營收且具穩定銷量（如 Apple）
- 高價值品牌：低銷量但高營收（如 Louis Vuitton）
- 流量型品牌：單價不高但依靠高銷量創造營收(如 小南瓜)
- 小眾品牌：營收與銷量規模相對較小

### 4. 平台營收結構偏向高價值導向
整體平台營收並非由大量低價商品支撐，而是由少數高價值品牌與高客單價類別主導。

---

## Business Insights 商業洞察

若以平台營運角度思考，本分析顯示：

- 平台應持續維持頭部高價值品牌合作，以穩定主要營收來源
- 可同時發展高銷量流量型品牌，以提升平台活躍度與商品滲透率
- 商品結構上，高客單價類別對平台營收的重要性明顯高於一般低價類別
